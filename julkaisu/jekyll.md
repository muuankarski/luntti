# Jekyll



Start a new site


```r
jekyll new site
cd site
jekyll serve
# => Now browse to http://localhost:4000
```


Asetukset ko. dokkariin


```r
jekyll serve --watch --baseurl''

# tai

jekyll serve --watch
```
